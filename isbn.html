ISBN

Para calcular el dígito de control para un ISBN de longitud 13, es necesario seguir los siguientes pasos:

  • Sumar los 12 primeros dígitos tras multiplar cada uno por el peso correspondiente (1 para las posiciones impares y 3 para las pares).
  • Realizar la operación módulo 10 a la suma anterior, si el resto es 0 hemos terminado.
  • Si el resto es distinto de 0, el dígito de control es: 10 - resto.

Para más información, consultar el manual oficial.

Ejemplo ISBN de 13 dígitos: 978-0-11-000222-4
